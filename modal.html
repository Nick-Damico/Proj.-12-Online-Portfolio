Photo Gallery

Project Description

In this project I created an interactive photo gallery using JavaScript and jQuery. The page is built using a mobile first, responsive design. The top of the page has a search area that filters out photos in the gallery for typed matching keywords.

A custom lightbox modal was built using JavaScript, no jQuery Plugins used. The lightbox appears on user click of thumbnails, the lightbox has photo description, and as to page through all photos in gallery. The animations are handled by Velocity.js without jQuery.

Interactive Video Player

Project Description

This project was built with JavaScript and jQuery using the HTML5 video, audio elements and their media API. The player synchronizes the video and transcript. The transcript is below the video and hightlights with video progress. When a user clicks any part of the transcript it takes them to the appropriate place in the video.

The custom video player has as for play, pause, closed captions, volumn off/on, and fullscreen display. There is also a progress bar using the progress HTML element which updates during play.

API Gallery

Project Description

The API Gallery project works with public API's from both Spotify and Flickr. This project uses jQuery and AJAX to request and display the returned data from those API's databases and displays it in an attractive gallery of images.

The gallery uses a jQuery Plugin lightbox when users click on thumbnail images from Flickr and Album artwork from Spotify. Flickr data includes title and descriptions of photos and Spotify includes artist names and album titles. Both Pages include as to sort and filter the data results in the gallery.

Site Optimization

Project Description

Apply the skills I learned for optimizing graphics, HTML, CSS, and JavaScript I was tasked with taking a poorly optimized page and reduce the total load size to under 1.3mb.

This project before optimization made over 43 server requests, had 11mb worth of files, and took over 20secs to load. Using best practices and tools such as, Chromes Developer panel and Gulp.js I made major reductions to those numbers. The final project makes only 22 requests, transfers 920kb, and loads within under 500ms;
